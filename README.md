# 🎯 Bull's Eye Target Detection Using Transfer Learning

## 📌 Overview

This project implements a computer vision model to automatically detect and classify Bull's Eye targets using **Transfer Learning** with **MobileNetV2**. By leveraging pre-trained ImageNet weights, the model achieves accurate image classification while significantly reducing training time and computational requirements.

---

## 🚀 Features

- ✅ Transfer Learning using MobileNetV2
- ✅ Data Augmentation for improved generalization
- ✅ Fine-tuning of pre-trained layers
- ✅ Performance Evaluation using multiple metrics
- ✅ Visualization of training and validation results
- ✅ Confusion Matrix and Classification Report
- ✅ ROC Curve and AUC Score Analysis

## 📊 Data Augmentation Techniques

To improve model robustness and prevent overfitting, the following augmentation techniques were applied:

- Rotation
- Zooming
- Width & Height Shifting
- Brightness Adjustment
- Horizontal Flipping

## 🎯 Objective

The primary objective of this project is to develop an efficient image classification system capable of accurately identifying Bull's Eye targets from image datasets. The project demonstrates the practical application of transfer learning in real-world computer vision tasks.
