# BULLS_EYE_PROJECT

Bull's Eye Target Detection Using Transfer Learning

Developed a computer vision model to automatically detect and classify Bull's Eye targets using Transfer Learning with MobileNetV2. The project leverages deep learning techniques to achieve accurate image classification while reducing training time through pre-trained ImageNet weights.

Key Features
Implemented MobileNetV2 as the base model for transfer learning.
Applied extensive data augmentation techniques such as rotation, zooming, shifting, brightness adjustment, and horizontal flipping to improve model generalization.
Fine-tuned the final layers of the network for better target detection performance.
Evaluated model performance using:
Accuracy and Loss Curves
Confusion Matrix
Classification Report (Precision, Recall, F1-Score)
ROC Curve and AUC Score
Built and tested the project using TensorFlow/Keras in Google Colab/Jupyter Notebook.
Technologies Used
Python
TensorFlow & Keras
MobileNetV2
NumPy
Matplotlib
Seaborn
Scikit-learn
Google Colab / Jupyter Notebook
Objective

The goal of this project is to develop an efficient and reliable image classification system capable of identifying Bull's Eye targets from image datasets, demonstrating the practical application of transfer learning in computer vision tasks.

Results

Achieved high classification performance by combining transfer learning, data augmentation, and fine-tuning techniques, making the model suitable for real-world target detection applications.
